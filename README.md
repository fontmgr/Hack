## Hack Font  
  
Requires:  

```shell
sudo bash -c "$(curl -LSs https://github.com/dfmgr/installer/raw/main/install.sh)"
```

Automatic:

```shell
bash -c "$(curl -LSs https://github.com/fontmgr/Hack/raw/main/install.sh)"
```

Install:

```shell
sudo fontmgr install Hack
```

Update:

```shell
sudo fontmgr update Hack
```
